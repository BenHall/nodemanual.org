A1: Third Party Modules

There are many third party modules for Node.js. The current master repository of modules is the wiki page, though you should also consider using the npm registry to find new ones.

This appendix is intended as a SMALL guide to new-comers to help them quickly find what are considered to be quality modules. It is not intended to be a complete list. There may be better more complete modules found elsewhere.

  • Module Installer: npm

  • HTTP Middleware: Connect

  • Web Framework: Express

  • Web Sockets: Socket.IO

  • HTML Parsing: HTML5

  • mDNS/Zeroconf/Bonjour

  • RabbitMQ, AMQP

  • mysql

  • Serialization: msgpack

  • Scraping: Apricot

  • Debugger:
    ndb is a CLI debugger
    inspector is a web based tool.

  • pcap binding

  • ncurses

  • Testing/TDD/BDD:
    vows,
    expresso,
    mjsunit.runner
