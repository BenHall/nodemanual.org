http.ClientResponse

[edit]
  • Events (3)
    • close
    • data
    • end
  • Functions (3)

    • setEncoding
  • Properties (4)

This object is created when making a request with http.request(). It is passed to the 'response' event of the request object.

The response implements the Readable Stream interface.
