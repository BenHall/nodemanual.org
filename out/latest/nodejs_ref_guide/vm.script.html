vm.Script

  • Functions (2)
    • runInNewContext
    • runInThisContext

This object is created as a result of the vm.createScript() method. It represents some compiled code than can be run at a later moment.

Methods

    • runInNewContext([Object sandbox])
      • String

    Similar to vm.runInNewContext(), this is a method of a precompiled Script object. ...

    Similar to vm.runInNewContext(), this is a method of a precompiled Script object.

       

    Arguments

    NameTypeDescription

    sandbox

    Object

    Optional. A global object with properties to pass into the Script object

    Description

    script.runInNewContext() runs the code of script with sandbox as the global object and returns the result. Running code does not have access to local scope.

    Running untrusted code is a tricky business requiring great care. To prevent accidental global variable leakage, script.runInNewContext() is quite useful, but safely running untrusted code requires a separate process.

    Returns

    A string representing the result of running code.
