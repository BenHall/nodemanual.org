fs.Stats

[edit]

Objects returned from fs.stat(), fs.lstat(), and fs.fstat() (and their synchronous counterparts) are of this type. The object contains the following methods:

For a regular file, util.inspect(fs.Stats) returns a string similar to this: 

{ dev: 2114,
  ino: 48064969,
  mode: 33188,
  nlink: 1,
  uid: 85,
  gid: 100,
  rdev: 0,
  size: 527,
  blksize: 4096,
  blocks: 8,
  atime: Mon, 10 Oct 2011 23:24:11 GMT,
  mtime: Mon, 10 Oct 2011 23:24:11 GMT,
  ctime: Mon, 10 Oct 2011 23:24:11 GMT }

Please note that atime, mtime, and ctime are instances of the Date object, and to compare the values of these objects you should use appropriate methods. For most general uses, getTime() returns the number of milliseconds elapsed since 1 January 1970 00:00:00 UTC, and this integer should be sufficient for any comparison. However, there are additional methods which can be used for displaying fuzzy information.
