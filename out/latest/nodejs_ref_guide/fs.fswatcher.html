fs.FSWatcher

  • Events (2)
    • change
    • error
  • Functions (1)
    • close

Objects returned from fs.watch() are of this type. You can monitor any changes that occur on a watched file by listening for the events in this object.

Events

    • change(String event, String filename)
      • Void

    Emitted when something changes in a watched directory or file. See more details in fs.watch(). ...

    filename

    String
