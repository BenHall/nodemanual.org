diffieHellman

  • Functions (8)
    • computeSecret
    • generateKeys
    • getGenerator
    • getPrime
    • getPrivateKey
    • getPublicKey
    • setPrivateKey

This class is a representation of the OpenSSL implementation of diffie-Hellman. It can be created as a returned value from crypto.createDiffieHellman().

For more information, you may want to read this chapter on using the crypto module.
